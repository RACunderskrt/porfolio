Où mon fioul

Mon premier projet universitaire en javascript, ce dernier devait utiliser des données présente sur une API. Ce site permet via une carte LeafLet voir les différentes stations essences autour d'une adresse donnée, en utilisant la base d'adresse nationale et d'y afficher les différents carburants et leurs prix. On peut aussi, si l'utilisateur l'accepte, utiliser sa position pour lancer la recherche.

Compétences

  • Développement orienté objet (c tou pt)
  • Récupération de données via une API
  • Interface web

Répihandi

Répihandi est une application mobile permettant a des bénévoles de se rendre disponible pour pouvoir encadrer des personnes en situation de handicap au sein d'une maison de répit. Le commanditaire de l'application est l'association "Cyril sa bataille" et cette application est réalisée au sein d'un projet universitaire pour l'IUT d'Arles.

Compétences

  • Développement orienté objet
  • Création d'une base de données PostgreSQL
  • Création et utilisation d'une API Rest avec ExpressJS
  • Interface web en VueJS
  • Interface mobile en Ionic
  • Travail en équipe avec la méthode SCRUM
  • Encryption des données
  • Respect des RGPD
  • Hébergement d'une application

WebHumsNeo

Lors de mes 2 stages réalisés pendant mon BUT, dans l'entreprise Airbus Helicopters en BUT2 et Scalian en BUT3, j'ai travaillé sur la maintenance et la création de fonctionnalité sur l'application WebHumsNeo. Cette application est utilisée par l'équipe HUMS d'Airbus Helicopters et permet en partie de gérer les différentes bases de données d'Airbus Helicopters sans avoir de connaissance en programmation/langage de base de données grâce à une interface graphique.

Compétences

  • Utilisation de NextJS
  • VueJS
  • Travail avec une équipe de professionnel
  • Répondre aux besoins d'un client non-informaticien
  • Réalisation de documentations techniques
